# חיבור ל-Vercel + Supabase — 5 דקות

אני לא יכול להתחבר לחשבונות שלך מרחוק.  
אבל **תיקנתי את הקוד** ויצרתי קובץ שמעלה הכל אוטומטית.

---

## מה תיקנתי עכשיו

- `vercel.json` בשורש הפרויקט — **מתקן את שגיאת 404**
- גם אם שכחת לבחור `client` ב-Vercel — זה אמור לעבוד

---

## מה אתה עושה — 3 שלבים בלבד

### שלב 1 — העלה קוד מעודכן ל-GitHub

לחץ פעמיים על הקובץ:

```
העלה-לגיטהאב.bat
```

(נמצא בתיקייה `D:\פרויקט ניסיון`)

הוא ישאל אותך את כתובת ה-GitHub — הדבק משהו כמו:
```
https://github.com/השם-שלך/reel-analyzer.git
```

---

### שלב 2 — Supabase (API)

1. היכנס לפרויקט שלך ב-[supabase.com](https://supabase.com)
2. **Edge Functions** → פרוס את הפונקציה `analyze`
3. **Project Settings** → **Edge Functions / Secrets**
   - הוסף: `OPENAI_API_KEY` = המפתח שלך

שמור בצד את:
- `Project URL`
- `anon public key`

---

### שלב 3 — Vercel (אתר)

1. היכנס ל-[vercel.com](https://vercel.com) → הפרויקט שלך
2. **Settings** → **Environment Variables**
   - `VITE_SUPABASE_URL` = ה-Project URL
   - `VITE_SUPABASE_ANON_KEY` = ה-anon public key
3. **Deployments** → על הדיפלוי האחרון → **⋯** → **Redeploy**

**Root Directory:**  
- אפשר **ריק** (השארנו vercel.json בשורש)  
- או `client` — גם בסדר

---

## בדיקה

פתח את Vercel → אמור לראות דף בית → **נתח סרטון** → העלה וידאו.

---

## עדיין 404?

ב-Vercel → **Settings** → **General** → **Root Directory**:

נסה **לרוקן** (Delete / ריק) → Save → Redeploy.

הקובץ `vercel.json` החדש בשורש אמור לבנות הכל אוטומטית.
