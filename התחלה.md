# איך מפעילים את Reel Analyzer

מדריך פשוט — בלי ידע טכני.

---

## שלב 1: התקן Node.js (פעם אחת)

1. היכנס ל-[nodejs.org](https://nodejs.org)
2. לחץ על הכפתור הירוק **LTS** והתקן
3. לחץ Next על הכל

---

## שלב 2: קבל מפתח OpenAI (פעם אחת)

1. היכנס ל-[platform.openai.com](https://platform.openai.com)
2. צור חשבון והוסף אשראי (כ-$5 מספיק להתחלה)
3. לך ל-**API Keys** → **Create new secret key**
4. העתק את המפתח (מתחיל ב-`sk-`)

---

## שלב 3: הדבק את המפתח

1. פתח את התיקייה: `D:\פרויקט ניסיון\server`
2. אם אין קובץ `.env` — העתק את `.env.example` ושנה שם ל-`.env`
3. פתח את `.env` בפנקס וכתוב:
   ```
   OPENAI_API_KEY=sk-הדבק-כאן-את-המפתח
   ```
4. שמור

---

## שלב 4: הפעל את האתר

1. לחץ **Windows + R**, כתוב `cmd`, Enter
2. העתק והדבק:
   ```
   cd /d "D:\פרויקט ניסיון"
   npm run dev
   ```
3. פתח בדפדפן: **http://localhost:5173**

---

- **"נתח סרטון"** — מעלים וידאו ומקבלים ניתוח

## בעיות נפוצות

| בעיה | פתרון |
|------|--------|
| `npm לא מזוהה` | התקן Node מחדש והפעל מחדש את המחשב |
