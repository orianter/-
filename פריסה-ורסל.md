# פריסה ל-Vercel + Render

האתר בנוי מ-**2 חלקים**:
- **Vercel** — דף הבית והממשק (מה שהלקוח רואה)
- **Render** — השרת שמנתח את הסרטון (חינם)

> Vercel לבד לא מספיק — ניתוח וידאו דורש שרת עם ffmpeg.

---

## חלק א׳ — העלה את הקוד ל-GitHub

### 1. צור חשבון ב-GitHub
[github.com](https://github.com) → Sign up (חינם)

### 2. צור Repository חדש
- לחץ **New repository**
- שם: `reel-analyzer`
- Private או Public — לפי בחירתך
- **Create repository**

### 3. העלה את הפרויקט

פתח **CMD** בתיקיית הפרויקט והרץ (החלף `YOUR_USERNAME`):

```bash
cd /d "D:\פרויקט ניסיון"
git init
git add .
git commit -m "Reel Analyzer initial"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/reel-analyzer.git
git push -u origin main
```

אם `git` לא מזוהה — התקן [Git for Windows](https://git-scm.com/download/win).

## חלק ב׳ — שרת הניתוח (Render) — קודם!

### 1. היכנס ל-[render.com](https://render.com)
הרשם עם GitHub (חינם).

### 2. New → Web Service
- בחר את ה-repository `reel-analyzer`
- **Name:** `reel-analyzer-api`
- **Root Directory:** `server`
- **Runtime:** Node
- **Build Command:** `npm install`
- **Start Command:** `node index.js`
- **Instance Type:** Free

### 3. Environment Variables (חשוב!)

לחץ **Add Environment Variable** והוסף:

| Key | Value |
|-----|--------|
| `OPENAI_API_KEY` | המפתח שלך מ-OpenAI (`sk-...`) |
| `CLIENT_URL` | נמלא אחרי Vercel — בינתיים השאר ריק |
| `NODE_ENV` | `production` |

### 4. Create Web Service
המתן 2–5 דקות. בסוף תקבל כתובת כמו:

```
https://reel-analyzer-api.onrender.com
```

**שמור את הכתובת הזו** — תצטרך אותה ל-Vercel.

בדיקה: פתח בדפדפן  
`https://reel-analyzer-api.onrender.com/api/health`  
אמור לראות: `{"ok":true,"hasApiKey":true}`

## חלק ג׳ — האתר (Vercel)

### 1. היכנס ל-[vercel.com](https://vercel.com)
הרשם עם GitHub (חינם).

### 2. Add New → Project
- בחר את `reel-analyzer`
- **Framework Preset:** Vite
- **Root Directory:** לחץ Edit → בחר `client`

### 3. Environment Variables

הוסף:

| Key | Value |
|-----|--------|
| `VITE_API_URL` | `https://reel-analyzer-api.onrender.com` |

(הכתובת מ-Render — **בלי** `/` בסוף)

### 4. Deploy
לחץ **Deploy**. תוך 1–2 דקות תקבל כתובת כמו:

```
https://reel-analyzer-xxxx.vercel.app
```

**זה הקישור שלך ללקוחות.**

### 5. עדכן את Render
חזור ל-Render → Environment → ערוך:

| Key | Value |
|-----|--------|
| `CLIENT_URL` | `https://reel-analyzer-xxxx.vercel.app` |

שמור — השרת ייטען מחדש.

## בעיות נפוצות

| בעיה | פתרון |
|------|--------|
| האתר נטען אבל ניתוח נכשל | בדוק `VITE_API_URL` ב-Vercel |
| `hasApiKey: false` | הוסף `OPENAI_API_KEY` ב-Render |
| ניתוח איטי בפעם הראשונה | Render חינמי "נרדם" — ממתין ~30 שניה |
| דף 404 ב-/analyze | וודא ש-`vercel.json` קיים בתיקיית `client` |
