# פריסה ל-Vercel + Supabase

האתר בנוי מ-**2 חלקים**:
- **Vercel** — דף הבית והממשק (מה שהלקוח רואה)
- **Supabase Edge Function** — API קטן שקורא ל-OpenAI

> בגרסה הזו הסרטון לא נשלח לשרת. Supabase מקבל רק פרטי טופס ומטא־דאטה בסיסי.

---

## חלק א׳ — העלה את הקוד ל-GitHub

### 1. צור חשבון ב-GitHub
[github.com](https://github.com) → Sign up (חינם)

### 2. צור Repository חדש
- לחץ **New repository**
- שם: `reel-analyzer`
- Private או Public — לפי בחירתך
- **Create repository**

### 3. העלה את הפרויקט

פתח **CMD** בתיקיית הפרויקט והרץ (החלף `YOUR_USERNAME`):

```bash
cd /d "D:\פרויקט ניסיון"
git init
git add .
git commit -m "Reel Analyzer initial"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/reel-analyzer.git
git push -u origin main
```

אם `git` לא מזוהה — התקן [Git for Windows](https://git-scm.com/download/win).

---

## חלק ב׳ — Supabase Function

### 1. היכנס ל-[supabase.com](https://supabase.com)
פתח את הפרויקט שיצרת.

### 2. פרוס את הפונקציה

מתיקיית הפרויקט:

```bash
supabase functions deploy analyze
```

### 3. Secret חשוב

ב-Supabase הוסף Secret:

| Key | Value |
|-----|--------|
| `OPENAI_API_KEY` | המפתח שלך מ-OpenAI (`sk-...`) |

### 4. שמור את פרטי ה-API

ב-Project Settings → API שמור:
- `Project URL`
- `anon public key`

---

## חלק ג׳ — האתר (Vercel)

### 1. היכנס ל-[vercel.com](https://vercel.com)
הרשם עם GitHub (חינם).

### 2. Add New → Project
- בחר את `reel-analyzer`
- **Framework Preset:** Vite
- **Root Directory:** לחץ Edit → בחר `client`

### 3. Environment Variables

הוסף:

| Key | Value |
|-----|--------|
| `VITE_SUPABASE_URL` | ה-Project URL |
| `VITE_SUPABASE_ANON_KEY` | ה-anon public key |

### 4. Deploy
לחץ **Deploy**. תוך 1–2 דקות תקבל כתובת כמו:

```
https://reel-analyzer-xxxx.vercel.app
```

**זה הקישור שלך ללקוחות.**

## עדכונים בעתיד

כל שינוי בקוד:
```bash
git add .
git commit -m "עדכון"
git push
```
Vercel יתעדכן **אוטומטית**. אם שינית Function, פרוס אותה שוב ל-Supabase.

## בעיות נפוצות

| בעיה | פתרון |
|------|--------|
| האתר נטען אבל ניתוח נכשל | בדוק `VITE_SUPABASE_URL` ו-`VITE_SUPABASE_ANON_KEY` ב-Vercel |
| חסר מפתח OpenAI | הוסף `OPENAI_API_KEY` ב-Supabase Secrets |
| הפונקציה לא עונה | ודא ש-`supabase functions deploy analyze` רץ בהצלחה |
| דף 404 ב-/analyze | וודא ש-`vercel.json` קיים בתיקיית `client` |
